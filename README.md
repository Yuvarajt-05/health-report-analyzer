# Health Report Analyzer
AI-based Health Report Analyzer and Chatbot system that analyzes medical reports and provides health insights using Machine Learning.

## Features

- Health Chatbot for symptom-based suggestions
- Medical Report Analyzer
- KNN Machine Learning Prediction
- Report Upload System
- Health Result Visualization

## Tech Stack

Frontend:
- React
- TypeScript
- Vite
- CSS

Backend:
- Python
- Flask
- SQLite Database

Machine Learning:
- KNN Algorithm
- Scikit-learn

## Project Structure
src/
components/
utils/
types/

backend/
app.py
ml_model.py
health_analyser.py


## How to Run Project

### Frontend
npm install
npm run dev


### Backend
cd backend
pip install -r requirements.txt
python app.py

How It Works:
User uploads medical report
System extracts health parameters
Machine Learning model analyzes report
Health condition prediction generated
Chatbot provides explanation and suggestions
